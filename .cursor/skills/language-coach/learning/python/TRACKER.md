# Python — learning tracker

**Read this file at the start of every Python language-coach session** before explaining a concept at length.

## How to use this table

| Column | Purpose |
|--------|---------|
| **Concept** | Short name |
| **Slug** | Filename under `concepts/` (no `.md`) |
| **Depth** | `none` \| `glance` \| `standard` \| `deep` — see skill reference |
| **Updated** | ISO date of last change to that concept |
| **Summary** | One line; enough to recall without opening the file |
| **Gap / next** | What is *not* yet covered or optional next step |

Add a row when a new concept is introduced. **Do not remove rows** when depth increases; update the row in place.
